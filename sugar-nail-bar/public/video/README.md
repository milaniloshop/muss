# Hero video

Drop the real salon hero video here as:

    hero-pinned.mp4

The Hero component (`src/components/Hero.tsx`) plays it full-bleed, muted,
autoplay + loop, with `public/images/hero.jpg` as the poster/fallback. If this
file is missing, the poster image shows through and nothing breaks.

Recommended: ~1080p, H.264 MP4, short seamless loop, compressed (< ~8 MB).
