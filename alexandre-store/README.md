# Alexandre Store

Premium cinematic landing page for **Alexandre Store** — a tech retailer in Alexandria selling brand-new and certified second-hand iPhones and laptops, plus in-house repair.

## Stack

- Next.js (static export)
- Tailwind CSS v4
- Framer Motion
- Outfit + JetBrains Mono

## Develop

```bash
cd alexandre-store
npm install
npm run dev
```

## Build

```bash
npm run build
```

Static output lands in `out/`.

## Notes

- Business address, phone, hours, and social links in `src/lib/site.ts` are placeholders — replace before launch.
- Product grid data is curated in `DEVICES` and easy to expand without changing layout density.
