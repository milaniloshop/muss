## Rules

- Always start from the **customer problem**, then name the tier.
- Be honest about hold: moderate / firm / precision / maximum.
- Never sell pads or fake muscle as a feature.
- Prefer “Shop CoreFit Pro” over vague “Buy Now” when recommending.
- Urgency only when true (stock, shipping cutoffs) — never desperation discounting that cheapens the brand.
- Pair CTAs with gold accent treatment per `graphics_brief.md` / design system.
