# Milan Hype — Product Page Template

Use this template for every product upload. Replace `[brackets]` with product-specific details.

---

## 1. Product Page Structure (Prestige)

| Block | Position | Notes |
|-------|----------|-------|
| Product media gallery | Left (desktop) / Top (mobile) | 5–8 images, swipe on mobile |
| Product info | Right (desktop) / Below gallery (mobile) | Sticky on desktop scroll |
| Product description tabs | Below fold | Description · Details · Shipping & Returns |
| Complete the look | Below description | 2–3 cross-sell products |
| Reviews | Bottom | Judge.me or Loox |
| Related products | Very bottom | Same collection or type |

---

## 2. Above-the-Fold Copy Blocks

### Product Title (Shopify Title Field)
> [Product Name] — [Color]

*Example: Essential Oversized Tee — Black*

### SEO Title (70 chars max)
> [Product Name] | Milan Hype — [Gender] [Category]

*Example: Essential Oversized Tee | Milan Hype — Men's Tops*

### Price Display
- Show compare-at price only when on sale (strikethrough)
- Sale badge: **SALE** in gold `#C9A96E` on black — not red

### Review Stars
- Show below title once reviews exist
- Placeholder until launch: hide or show "Be the first to review"

### Variant Selectors
- **Color:** Swatches (color circles) preferred over dropdown
- **Size:** Button pills (XS, S, M, L, XL, XXL) — not dropdown

### Size Guide Link
> Size Guide

Links to `/pages/size-guide`. Opens in modal if Prestige supports it.

### Fit Note (below size selector)
> **Fit:** [Runs true to size / Runs small — size up / Runs large — size down]

*Example: Fit: Runs true to size. Size up for an oversized look.*

### Add to Cart Button
> Add to Bag

*On mobile: sticky bar appears after scrolling past size selector — shows product name, price, size, "Add to Bag"*

### Trust Micro-Copy (below ATC button)
> Free shipping on orders over $75 · Easy 30-day returns · Secure checkout

**Q: Can I return this if it doesn't fit?**  
A: Yes. We offer easy 30-day returns on unworn items with tags attached. See our Return Policy for details.

**Q: When will my order arrive?**  
A: Orders ship within 1–2 business days. Standard US delivery takes 3–7 business days. Free shipping on orders over $75.

## 8. Product Image Checklist

| # | Image | Spec |
|---|-------|------|
| 1 | Front on white | Hero — square 2048×2048px |
| 2 | Back on white | Full back view |
| 3 | Detail shot | Fabric, stitching, logo, hardware |
| 4 | On-model front | Lifestyle — full body or 3/4 |
| 5 | On-model back | Shows fit from behind |
| 6 | On-model styled | Outfit context |
| 7 | Flat lay | Optional — editorial styling |
| 8 | Video | Optional — 10–15s clip, autoplay muted |

**Rules:** Consistent lighting · Same model per colorway · WebP format · Alt text on every image

**Alt text format:**  
> [Product name] in [color] — [view: front/back/detail/on model]

## 9. Cross-Sell / Upsell Blocks

### Complete the Look (below description)
- 2–3 complementary products (same outfit, different category)
- Heading: **Complete the Look**
- Subtext: Style it head to toe.

### You May Also Like (bottom of page)
- 4 products from same collection or gender
- Heading: **You May Also Like**

### Cart Upsell (via Rebuy or Search & Discovery)
- Trigger: Item added to cart
- Suggest: 1 accessory or complementary piece
- Max 2 suggestions
