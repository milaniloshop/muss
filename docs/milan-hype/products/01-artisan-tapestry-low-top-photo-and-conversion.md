## How to Shoot the Missing Photos (DIY Premium)

You don't need a studio. You need consistency.

### Setup (Under $30)
- **Background:** White poster board or seamless paper against a wall
- **Light:** Shoot near a large window (10am–2pm) — no direct sun, no overhead yellow bulbs
- **Phone:** Clean lens, grid on, 2x zoom for detail shots (reduces distortion)
- **Prop:** Remove the black glove from product shots — it reads "reseller" not "fashion brand"
- **Floor shots:** Don't use wood floor for hero images — it cheapens premium perception

### Shot order (30 minutes)
1. White background — 8 angles (hero set)
2. Macro — fabric + pins (move phone closer, tap to focus)
3. On-foot — 3 outfit changes max, same location
4. Box shot — 2 minutes

### Editing
- Brightness: slight lift
- Shadows: soften
- Crop: square 2048×2048px for Shopify
- Export: WebP or JPG 85% quality
- **Do not** over-filter — keep colors true to product

### Description Structure (Scannable on Mobile)

1. **One-line hook** — bold
2. **Why You'll Love It** — 5 bullet points (already written)
3. **Details** — spec table
4. **Style It** — 2 sentences
5. **Shipping & Returns** — 3 bullets

**Do not** wall-of-text. Mobile customers scroll in 8 seconds.

### Reduce Returns (Footwear = High Return Category)

1. **Size guide page** — link on every footwear product
2. **Fit note on page** — "True to size — half size up for wide feet"
3. **On-foot photos** — mandatory before scaling ads
4. **FAQ on product page** — fit + care questions answered inline
5. **Interior lining photo** — shows quality, reduces "is this legit" anxiety

### Urgency (Premium Style — Not Tacky)

✅ Use:
- "Limited run"
- "Only [X] pairs left"
- "New arrival"

❌ Avoid:
- Flashing countdown timers
- "BUY NOW!!!" red banners
- Fake scarcity

### Page Speed

- Compress all images to < 300KB each
- Max 10 images on product page
- Lazy-load images below fold
