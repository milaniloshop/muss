## Mobile Optimization Notes

1. **Hero:** Single CTA primary ("Shop Women"), secondary as text link
2. **Collections:** 2-column grid, large tap targets
3. **Products:** 2-column grid, quick-add visible without hover
4. **Trust bar:** 2×2 grid, short text
5. **Newsletter:** Full-width input and button
6. **Sticky header:** Logo center or left, cart icon right, hamburger left
7. **Font sizes:** H1 min 32px, body 16px, no input zoom on iOS
