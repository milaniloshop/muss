# Chuckle & Bliss

Premium cinematic landing page for **Chuckle & Bliss**, a women's boutique in the Gaslamp Shoppes at 54 E Granada Blvd, Ormond Beach, FL.

## Run locally

```bash
cd chuckle-bliss
npm install
npm run dev
```

Open http://localhost:3000

## Build static export

```bash
cd chuckle-bliss
npm run build
```

Output lands in `chuckle-bliss/out/` (GitHub Pages / any static host).

## Content notes

- Owner story and naming (Carter & Bryson → Chuckle & Bliss) from Ormond Beach Observer coverage
- Hours based on the Granada boutique space listing (Mon–Fri 10–6, Sat 10–5)
- Instagram gallery uses curated warm-graded imagery linked to [@chuckleandbliss](https://www.instagram.com/chuckleandbliss/); swap in SnapWidget / Behold / Elfsight for a live feed via the marked slot in `ShopTheLook`
- Reviews section presents editorial 5★ guest quotes styled for the perfect local reputation — replace with live Google review text when available
